# Soul Memory Rules

## 可沉淀为长期记忆

- 饮食禁忌：不吃辣、不吃香菜、素食、清真、过敏等。
- 预算偏好：默认总预算上限、酒店单晚上限、是否偏好性价比。
- 交通偏好：地铁优先、少打车、自驾优先、少步行。
- 是否喜欢拍照：出片、机位、日落、夜景、拍照优先。
- 是否喜欢安静：人少、小众、避开人群、低社交压力。
- 是否带老人/孩子：长辈、爸妈、亲子、带娃、推车等。
- 排队容忍度：不排队、最多排队几分钟、可接受排队时长。
- 步行强度：少走路、中等步行、暴走、多景点高强度。
- 常去城市：用户明确提到“常去/经常去/常玩”的城市。
- 常用人格模式：松弛感、特种兵、社恐、家庭、学生、长辈、出片党、美食脑袋。
- 旅行节奏：松弛、特种兵、家庭、长辈友好、出片、美食优先。
- 住宿偏好：近地铁、安静、干净、安全、不要民宿等。

## 使用规则

- 本轮用户明确表达 > Soul 长期记忆 > 表单默认值 > 系统默认值。
- 记忆只能辅助排序和默认值，不得强行改变用户本轮要求。
- 涉及健康、过敏、预算上限、交通偏好时必须优先尊重。
