# 异常处理

| 错误码 / 现象 | 展示给用户 |
|---|---|
| 登录失效 / `status: error` / `AUTH_REQUIRED` / `AUTH_FAILED` | Token 不存在或已失效，请重新触发登录，我会展示新的授权二维码 |
| `10001` 城市未开通 | 该城市暂未开通服务，建议使用美团 App 确认是否支持 |
| `10002` 未知错误 | 信息异常，请检查下单参数。帮忙类用 businessType=1，帮买用 2 |
| `10112` 区域未开通 | 该地址所在区域暂未开通服务，建议更换到其他地址 |
| `10113` 城市未开通 | 该地址所在城市暂未开通服务，建议切换到其他城市 |
| `10114` 因恶劣天气/道路管制等暂无法配送 | 因恶劣天气/道路管制/疫情等原因，暂无法配送，敬请谅解，建议稍后重试 |
| `10115` 配送距离过远 | 配送距离过远，请重新选择地址 |
| `10116` 无效地址 | 请选择有效的地址，提供更精确的地址（含街道/楼号/校区名）有助于定位 |
| `10309` 配送费已动态发生变化 | 配送费已变动，请重新下单（同一次对话内重新预览即可） |
| `10311` orderToken 失效 | 订单token失效，请重新下单（同进程内完成） |
| `10401` 配送费计算失败 | 配送费计算失败，请重试一次 |
| `11004~11008` 风控拒绝 | 下单信息异常，请检查地址和备注信息后重试 |
| POI 搜索无结果 | 未找到该地址，请提供更精确的地址（含街道/楼号） |
| 费用预览失败 | 费用预览失败，请检查地址坐标是否正确或尝试缩短距离 |
| 网络超时 | 网络请求超时，请检查网络后重试 |
| `响应解析失败` | 接口响应异常，请检查网络/VPN 连接后重试 |
